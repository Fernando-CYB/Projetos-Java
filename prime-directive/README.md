# Prime Directive - Java

Projeto desenvolvido em Java com foco em lógica de programação e uso de loops para identificação de números primos.

## Descrição
Este projeto tem como objetivo identificar números primos e filtrar apenas os valores primos de um conjunto de números.

Um número primo é um número inteiro maior que 1 que é divisível apenas por 1 e por ele mesmo.

## Funcionalidades
- Verificação se um número é primo
- Filtragem de números primos a partir de um array
- Armazenamento dos números primos em uma lista (`ArrayList`)

## Conceitos Praticados
- Estruturas de repetição (`for`)
- Condicionais
- Métodos
- Arrays
- ArrayList
- Lógica matemática
- Programação Orientada a Objetos
