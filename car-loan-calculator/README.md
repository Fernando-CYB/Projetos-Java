# Car Loan Calculator - Java

Calculadora simples de pagamento mensal de empréstimo de carro desenvolvida em Java.

## Descrição
Este projeto simula o cálculo do pagamento mensal de um empréstimo de carro com base em:

- Valor total do empréstimo
- Taxa de juros anual
- Duração do empréstimo (em anos)
- Valor de entrada (down payment)

O programa utiliza condicionais, expressões booleanas e operações aritméticas para validar os dados e calcular o valor final da parcela mensal.

## Funcionalidades
- Validação de empréstimo inválido
- Verificação se o carro pode ser pago à vista
- Cálculo do valor restante após entrada
- Cálculo do pagamento mensal com juros

## Conceitos praticados
- Estruturas condicionais (`if / else`)
- Operadores lógicos e booleanos
- Operações aritméticas
- Variáveis e tipos primitivos
- Lógica de programação

## Como executar
- Compile e execute o arquivo CarLoan.java
- Exeute pelo terminal ou IDE
