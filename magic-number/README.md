# Magic Number – Java

Projeto desenvolvido durante meus estudos em Java que demonstra
um truque matemático utilizando operações aritméticas.

##  Conceitos praticados
- Variáveis
- Operadores aritméticos
- Ordem das operações
- Estrutura básica em Java

##  Como executar
1. Compile o arquivo `Magic.java`
2. Execute o programa pelo terminal ou IDE
