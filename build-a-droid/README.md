# Build a Droid - Java

Projeto desenvolvido em Java com foco em Programação Orientada a Objetos.

## Descrição
Simulação de droides que executam tarefas, consomem energia, transferem bateria entre si e podem ser recarregados.

## Conceitos Praticados
- Classes e Objetos
- Construtores
- Métodos
- Encapsulamento
- Manipulação de estado
- Interação entre objetos

## Funcionalidades
- Execução de tarefas com consumo de bateria
- Relatório de nível de energia
- Transferência de energia entre droides
- Recarga completa de bateria

## Como Executar
- Compile e execute o arquivo `Droid.java`
- Exeute pelo terminal ou IDE
