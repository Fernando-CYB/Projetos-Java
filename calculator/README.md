# Calculator - Java

Calculadora aritmética simples desenvolvida em Java utilizando os conceitos de Programação Orientada a Objetos (POO).

## Funcionalidades
- Adição
- Subtração
- Multiplicação
- Divisão
- Operação de módulo (resto da divisão)

## Conceitos Praticados
- Classes e Objetos
- Métodos
- Encapsulamento
- Operadores aritméticos
- Tratamento de exceções

## Como Executar
- Compile e execute o arquivo `Calculator.java`
- Exeute pelo terminal ou IDE
